# 알고리즘 뿌셔보자!

## 스터디 방식

- 현 레포지토리를 자신의 개인 레포지토리로 fork한다.
- 백준허브 등에 연동해 자동으로 업로드되는 확장 프로그램을 사용해 자신의 개인 레포지토리에 자동 commit 되도록 설정한다.
- 현 레포지토리의 자신의 브랜치를 만들고, 그 브랜치에 pull request를 올린다.
- 다른 스터디원이 approve 후 merge 한다.
